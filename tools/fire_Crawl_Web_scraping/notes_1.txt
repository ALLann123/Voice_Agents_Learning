====What is Firecrawl=====
- Is an LLM-Optimized web reader
- Normal web pages contain:
	> HTML
	> CSS
	> JS
	> Ads
	> Navigation bars
	> Cookie Banners
	> Sidebars
	> Comments
- An LLM doesnt need any of that.

=====Main Features======
- Crawl a web site like:
	"https://docs.python.org/"
- Firecrawl reads the page:
	firecrawl.scrape_url(url)
Returns something like:

============Python Example=========
#!/usr/bin/python3
import os
from dotenv import load_dotenv
from firecrawl import FirecrawlApp

# environment varibles to be loaded
load_dotenv()

# get our API
API_KEY=os.getenv("FIRECRAWL")

# create an instance
app=FirecrawlApp(api_key=API_KEY)

# get the response
response=app.scrape_url(
    url="https://docs.python.org/3/tutorial/",
    formats=["markdown"]
)

print(response)
